◆ readLux()

float Adafruit_MAX44009::readLux ( )

Read lux value with full 12-bit resolution.

Uses a sequential 2-byte read starting at LUX_HIGH to ensure consistent data. The sensor disables internal register updates during I2C transactions.

Returns
Lux value as float, or NAN if sensor is in overrange condition
