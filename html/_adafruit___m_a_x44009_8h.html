Detailed Description

Arduino library for the MAX44009 ambient light sensor.

This is a library for the MAX44009 ambient light sensor with I2C interface. The MAX44009 provides wide dynamic range (0.045 to 188,000 lux) with ultra-low power consumption.

enum max44009_integration_time_t

Integration time settings for the MAX44009.

In automatic mode (MANUAL=0), only the first four values (800ms-100ms) are available. The shorter integration times (50ms-6.25ms) require manual mode to be enabled.

◆ max44009_mode_t

enum max44009_mode_t

Operating mode settings for the MAX44009.

Controls the CONT (bit 7) and MANUAL (bit 6) bits of the CONFIG register.

  • DEFAULT: Auto-ranging, measures every 800ms (lowest power)
  • CONTINUOUS: Auto-ranging, measures as fast as integration time allows
  • MANUAL: Manual gain/time control, measures every 800ms
  • MANUAL_CONTINUOUS: Manual gain/time, fast updates
